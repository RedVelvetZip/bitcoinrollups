Verification

None

Bitcoin does not verify Rootstock transactions.

Consensus

Merge-mining

Rootstock is merge-mined.

Bitcoin Hashpower

None

Rootstock transaction finality and block ordering is secured by 0% of Bitcoin hashpower.

Alternative Token

None

Rootstock does not have a custom token.

Technology

State correctness

Rootstock miners enforce state correctness.

Verification

Rootstock miners are solely responsible for verificaiton.

Data availability

Rootstock uses Bitcoin for data availability.

Operator

Open

Block production is permissionless. Anyone who is willing to merge-mine can produce blocks and earn rewards. If interested parties do not possess enough capital to compete in the mining market, they can join a mining pool.

Withdrawals

Reliant on Rootstock signers.

Withdrawals are currently permissioned by a federated group of signers. Users must trust, that when they deposit BTC into the Roostock blockchain, the signers will not collude and steal their BTC.

Source Code

Open-source

Rootstock is open-source.
